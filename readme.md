# Bootstrap Portfolio

This is my personal **Bootstrap Portfolio** project. I created this project to practice and showcase my skills with **HTML, CSS, and Bootstrap**, and to build a responsive, modern portfolio layout.

## Features

- Fully responsive design using **Bootstrap 5**
- Clean and modern layout
- Sections include:
  - About Me
  - Skills
  - Projects
  - Contact Form
- Custom CSS for personalized styling

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Optional: JavaScript (for interactive elements)
